# Arduino UNO Interfacing with Ultrasonic Sensor (HC-SR04)

## Overview
This project demonstrates how to interface an HC-SR04 Ultrasonic Sensor with an Arduino UNO to measure the distance of an object. The measured distance is displayed on the Serial Monitor in centimeters.

## Components Required
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Jumper Wires
- USB Cable

## Circuit Connections

| HC-SR04 Pin | Arduino UNO Pin |
|-------------|-----------------|
| VCC         | 5V              |
| GND         | GND             |
| TRIG        |
